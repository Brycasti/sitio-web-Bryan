10 Etiquetas en HTML

1. Video

El elemento HTML incorpora un reproductor multimedia que permite reproducir videos en el documento. También se puede utilizar para contenido de audio, pero el elemento puede proporcionar una experiencia de usuario más adecuada.

2. Audio

El elemento HTML se utiliza para incrustar contenido de sonido en los documentos. Puede contener una o más fuentes de audio, representadas mediante el atributo o el elemento: el navegador elegirá la más adecuada.

3. iFrame

El elemento HTML representa un contexto de navegación anidado , que incorpora otra página HTML

4. Input

El elemento HTML se utiliza para crear controles interactivos para formularios basados ​​en la web con el fin de aceptar datos del usuario; hay una amplia variedad de tipos de datos de entrada y widgets de control disponibles, según el dispositivo y el agente de usuario . El elemento es uno de los más potentes y complejos de todo HTML debido a la gran cantidad de combinaciones de tipos de entrada y atributos.

5. Legend

El elemento HTML representa un título para el contenido de su elemento padre .
Choose your favorite monster

6. Mark

El elemento HTML representa texto que está marcado o resaltado para fines de referencia o notación debido a la relevancia del pasaje marcado en el contexto circundante.

Search results for "salamander":

Several species of salamander inhabit the temperate rainforest of the Pacific Northwest.

Most salamanders are nocturnal, and hunt for insects, worms, and other small creatures.

7. Ins

El elemento HTML representa un intervalo de texto que se ha añadido a un documento. Puedes utilizar el elemento para representar de forma similar un intervalo de texto que se ha eliminado del documento.

“You're late!”

“I apologize for the delay.”

“A wizard is never late …”

8. Summary

El elemento HTML especifica un resumen, un título o una leyenda para el cuadro de divulgación de un elemento. Al hacer clic en el elemento, se alterna entre el estado abierto y cerrado del elemento principal .
I have keys but no doors. I have space but no room. You can enter but can’t leave. What am I? A keyboard.

9. Var

El elemento HTML representa el nombre de una variable en una expresión matemática o en un contexto de programación. Normalmente se presenta con una versión en cursiva del tipo de letra actual, aunque ese comportamiento depende del navegador.

The volume of a box is l × w × h, where l represents the length, w the width and h the height of the box.

10. Progress

El elemento HTML muestra un indicador que muestra el progreso de finalización de una tarea, normalmente mostrado como una barra de progreso.
